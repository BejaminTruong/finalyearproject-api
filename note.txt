- Trong package.json lúc deploy heroku thì chạy lệnh này:
    "build": "npm run create-build && npm run build-babel"

- Dưới môi trường dev thì trường hợp đã có thư mục build thì chạy lệnh này còn chưa có thì phải tạo tay thư mục build rồi mới chạy được lệnh
như ở trên:
    "build": "npm run clean && npm run build-babel"
